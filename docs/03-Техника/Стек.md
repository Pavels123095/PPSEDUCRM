# Стек

> [!info] Технологии PPSEDUCRM v1

## Backend

| Компонент | Версия / пакет |
|-----------|----------------|
| PHP | 8.2+ |
| Laravel | 11 |
| Auth | Laravel Sanctum 4 |
| RBAC | spatie/laravel-permission 6 |
| БД | MySQL 8 (prod), SQLite (dev) |
| Очереди | Redis |
| Локаль | `ru`, timezone `Europe/Moscow` |

## Frontend

| Компонент | Версия / пакет |

| TypeScript | да |

| Роутинг | React Router |
| Данные | TanStack Query |
| Стили | Tailwind CSS |
| HTTP | axios / fetch |

| Сервис | Назначение |
|--------|------------|

| Mailpit | Тестовая почта (порт 8025) |

## Стандарты РФ (v1)

> [!info] rf-standards
> - Локаль `ru`, формат дат `dd.MM.yyyy`
> - Валидация **СНИЛС** (`SnilsRule`, контрольная сумма)
> - Паспорт РФ (серия/номер)
> - Аудит-лог критичных действий (`audit_logs`)
> - Задел под ФИС ФРДО: `compliance_metadata` (JSON, nullable)

## Адаптивные breakpoints

| Breakpoint | Поведение UI |
|------------|--------------|
| <768px (mobile) | Bottom nav, карточки |
| 768–1024px (tablet) | Collapsible sidebar |
| >1024px (desktop) | Полный sidebar + таблицы |
