# Модуль — Педагоги

> [!info] Назначение
> Расписание занятий, управление аудиториями, учёт отработанных часов и отчёты.

## Сущности

- **Teacher** — ставка, дисциплины, `user_id`
- **Classroom** — номер, корпус, вместимость, оборудование
- **ScheduleSlot** — дата/время, тип (лекция/лаб/консультация), `teacher_id`, `classroom_id`
- **WorkSession** — фактические часы, тип активности, связь со слотом

## Типы занятий

| Enum | Русское название |
|------|------------------|
| `lecture` | Лекция |
| `lab` | Лабораторная |
| `consultation` | Консультация |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET/POST | `/api/schedule-slots` | CRUD расписания |
| GET/POST | `/api/classrooms` | Аудитории |
| GET | `/api/classrooms/availability` | Свободные слоты |
| GET/POST | `/api/work-sessions` | Учёт часов |
| GET | `/api/teachers/{id}/hours-report` | Отчёт за период |

## Конфликт-детекция

> [!info] ScheduleConflictService
> При создании/обновлении слота проверяется пересечение по аудитории и педагогу на тот же интервал времени.

## UI (teacher)

- Календарь / неделя — расписание
- Аудитории — список и доступность
- Форма часов — ввод `work_sessions`
- Отчёт — сумма по типам за период
