# Модуль — Интеграция 1С

> [!info] Назначение
> Подготовка обмена данными с 1С: внешние идентификаторы, webhook, export/import, фоновые jobs, логирование.

## Поля синхронизации

На сущностях `applicants`, `students`, `contracts`, `teachers`:

| Поле | Тип | Описание |
|------|-----|----------|
| `external_id` | string | ID в 1С |
| `sync_status` | enum | `pending`, `synced`, `error` |
| `last_synced_at` | datetime | Время последней синхронизации |

## API (role: admin)

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/integrations/1c/webhook` | Приём событий от 1С |
| GET | `/api/integrations/1c/export/{entity}` | Выгрузка JSON |
| POST | `/api/integrations/1c/import` | Загрузка пакета |

## Фоновые задачи

- `SyncApplicantTo1C` — отправка абитуриента в 1С
- `ImportStudentsFrom1C` — импорт студентов из 1С

## Логирование

Таблица `integration_logs` — все входящие/исходящие операции с payload и статусом.

## Конфигурация

```env
INTEGRATION_1C_WEBHOOK_SECRET=change-me-in-production
```

Файл `config/integrations.php` — настройки сущностей и маппинг полей.
