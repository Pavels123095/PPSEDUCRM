# Этап 09 — Адаптивный UI

> [!todo] Цель этапа
> Единый responsive layout, мобильная навигация, русская локализация.

## Задачи

- [ ] `ResponsiveLayout` — breakpoints mobile / tablet / desktop
- [ ] Mobile (<768px): bottom navigation, карточки вместо таблиц
- [ ] Tablet (768–1024px): collapsible sidebar
- [ ] Desktop (>1024px): полный sidebar + таблицы с фильтрами
- [ ] `RoleLayout` — меню по роли (manager / teacher / student / admin)
- [ ] TanStack Query hooks в `frontend/src/api/`
- [ ] `i18n/ru.json` — все тексты на русском
- [ ] Формат дат `dd.MM.yyyy`, часовой пояс `Europe/Moscow`

## Структура frontend

```
frontend/src/
├── api/           # axios + TanStack Query hooks
├── components/    # Button, Table, Card, ScheduleGrid
├── layouts/       # ResponsiveLayout, RoleLayout
├── pages/
│   ├── manager/
│   ├── teacher/
│   ├── student/
│   └── admin/
├── routes/        # ProtectedRoute по роли
└── i18n/          # ru.json
```

## Страницы по ролям

| Роль | Страницы |
|------|----------|
| Manager | Dashboard, абитуриенты, договоры |
| Teacher | Расписание, аудитории, часы, отчёт |
| Student | Расписание, профиль |
| Admin | Пользователи, справочники, логи 1С |

## Критерий завершения

> [!success] Готово, когда
> Все ролевые страницы работают на трёх breakpoints; тексты и даты на русском; навигация интуитивна на телефоне.
