# Этап 08 — Интеграция 1С

> [!todo] Цель этапа
> Инфраструктура обмена с 1С: `external_id`, webhook/export/import, queue jobs, логи.

## Задачи

- [ ] Поля `external_id`, `sync_status`, `last_synced_at` на `applicants`, `students`, `contracts`, `teachers`
- [ ] `POST /api/integrations/1c/webhook` — приём событий (stub + валидация подписи)
- [ ] `GET /api/integrations/1c/export/{entity}` — выгрузка JSON
- [ ] `POST /api/integrations/1c/import` — загрузка пакета
- [ ] Jobs: `SyncApplicantTo1C`, `ImportStudentsFrom1C`
- [ ] Таблица `integration_logs` + UI admin/logs

## Конфигурация

- `config/integrations.php`
- `.env`: `INTEGRATION_1C_WEBHOOK_SECRET`

## Ограничения v1

> [!info] Только инфраструктура
> Реальный обмен потребует спецификации вашей конфигурации 1С (УТ, БГУ, ЗУП и т.д.). В v1 — API-заглушки с валидацией и логированием.

## Критерий завершения

> [!success] Готово, когда
> Admin видит логи интеграции; API 1С принимает и отдаёт JSON с `external_id`; webhook проверяет подпись.
