# Этап 07 — Кабинет студента

> [!todo] Цель этапа
> Read-only API и UI личного кабинета: расписание, профиль, уведомления.

## Задачи

- [ ] `GET /api/student/schedule` — пары на день/неделю с аудиторией и временем
- [ ] `GET /api/student/profile` — группа, курс, контакты
- [ ] `GET /api/student/notifications` — список уведомлений
- [ ] Middleware `role:student` на всех student-эндпоинтах
- [ ] Frontend: расписание на сегодня/неделю, детали пары, профиль

## Страницы UI

| Страница | Описание |
|----------|----------|
| Расписание (день) | Пары сегодня: время, аудитория, педагог |
| Расписание (неделя) | Сетка или список на 7 дней |
| Профиль | ФИО, группа, курс |
| Уведомления | Список с датами |

## Критерий завершения

> [!success] Готово, когда
> Студент видит свои пары с временем и аудиторией на телефоне и десктопе (адаптивный layout из [[Этап 09 — Адаптивный UI]]).
