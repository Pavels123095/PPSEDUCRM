# Этап 06 — Часы педагогов

> [!todo] Цель этапа
> Учёт отработанных часов (`work_sessions`) и отчёты по периоду.

## Задачи

- [ ] `WorkSessionController` — создание записей часов (привязка к слоту или ручной ввод)
- [ ] `GET /api/teachers/{id}/hours-report` — отчёт: лекции / лабы / консультации
- [ ] Валидация: часы > 0, тип занятия из enum
- [ ] Frontend: форма ввода часов, таблица отчёта за период

## Типы активности

| Тип | Описание |
|-----|----------|
| `lecture` | Лекционные часы |
| `lab` | Лабораторные часы |
| `consultation` | Консультации |

## API-эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET/POST | `/api/work-sessions` | CRUD учёта часов |
| GET | `/api/teachers/{id}/hours-report` | Отчёт за период |

## Критерий завершения

> [!success] Готово, когда
> Педагог вносит часы по типам занятий; отчёт за месяц суммирует лекции, лабы и консультации отдельно.
