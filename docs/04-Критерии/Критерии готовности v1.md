# Критерии готовности v1

> [!info] Версия 1
> Полная система для одного учреждения: все 4 модуля, адаптивный web, Tauri desktop, подготовка 1С.

## Функциональные критерии

- [ ] **Менеджер** ведёт абитуриента от «новый» до «договор подписан» через UI
- [ ] **Педагог** видит расписание, аудитории, вносит отработанные часы по типам занятий
- [ ] **Студент** видит свои пары с временем и аудиторией на телефоне и десктопе
- [ ] **Admin** видит логи интеграции; API 1С принимает/отдаёт JSON с `external_id`
- [ ] **Tauri desktop** открывает тот же интерфейс, что и браузер
- [ ] Все тексты и даты на русском; **СНИЛС** валидируется
